denoiser.viz.plots module#

denoiser.viz.plots.carpet_plot(arr, unfold='classic', colorbar=True, transpose=False, figsize=None)[source]#

Carpet Plot of ND array.

The last dimension if plotted on the horizontal axis, The other are flattened on the vertical one. :param arr: The ND array to plot. :type arr: numpy.ndarray :param unfold: The method to unfold the (N-1)D first dimensions.

If “classic” the data is simply row-wise unfolded If “zigzag” (only for 2D spatial dimensions), the data is unfolded following a diagonal zig-zag pattern starting from top left.

Parameters:

  • colorbar (bool) – flag to show a colorbar

  • transpose (bool) – If true, switch the horizontal and vertical axis.

Returns:

The plotted figure.

Return type:

matplotlib.pyplot.figure

Flatten a 2D array with a zigzag pattern.

Parameters:

Returns:

A flattened 1D array.

Return type:

numpy.ndarray

Notes

Flattenning a 2D array with a zigzag pattern preserves a spatial contiguity in the flattened version, unlike the classical row or column-wise stacking.
